# card_games_cli
A card game system cli made in Rust for fun. 
Originally made by my around the winter of 2024.

The current supported games are go fish and war. I was working on blackjack before losing interest. 

The system uses a main menu where users can choose to play a game, change a "card set", or quit. The "card set" is the "skin" of the cards and are defined like the .txt files in the card_sets folder.
